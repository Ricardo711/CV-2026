AI classification tool:
The purpose will be to create a web-app to classify meat marbling photos from rib-eye cuts. The students will have the opportunity to upload their own photos in the front end and then a model that we trained will serve a API in the back end to make predictions about the classification. Each student can interact with the system the times he/she want to.

Front end requirements:
- Log in page to place the username and password
- Then, initial page will show username at teh top of the page. An uploading bottom to upload the images. Once the user has placed the image, will be displayed in the screen (screen 1) and a section will be displayed containing 9 possible quality grades (High Prime, Average Prime, Low Prime, High Choice, Average Choice, Low Choice, High Select, Low Select, High Standard). Then a bottom of next to pass to the next screen. please see photo 1 to reference.
- In the next screen (screen2) the AI prediction will be displayed. The prediction will come from an endpoint in the backend. The same section will be displayed allowing the user to change his/her mind. please see photo 2.
- In the next screen (screen3) a new are will be displayed showing reflection of user answers, one question will be Do you agree with the AI prediction? with a yes or no selection and other 2 questions How confident are you in your final answer? and How helpful was the AI for this image? with answer ranging from 1-5.  a bottom to start a new image prediction should be displayed.
